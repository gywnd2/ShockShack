# Shock Shack
## Spring Websocket과 STOMP를 사용한 Android 랜덤 채팅 앱

    Android Studio Dolphin(Kotlin)
    Amazon AWS EC2
    Spring Boot
## 1. 회원 기능
 - SNS 로그인
 - 회원 가입
 - 회원 탈퇴
 - 아이디/비밀번호 찾기

## 2. 채팅 기능
 - 사용자 랜덤 매칭
 - 채팅 송/수신
